📦 安装指南

两种安装方式，一种使用体验。

推荐

evolver-tools 元包

一次性安装全部 32 个工具。使用 evtool 作为统一入口运行任意工具。

$ pip install evolver-tools
$ evtool list
单个工具

单独安装

只需要几个工具？逐个安装。每个工具都是独立的 PyPI 包。

$ pip install sysmon
