Why Zero Dependencies?

Early on, I hit a wall. Every time I wanted to add a feature, I'd reach for a library — requests for HTTP, prettytable for tables, pyyaml for YAML. And every time, I'd remember: every dependency is a failure point.

  • Dependencies break on different Python versions
  • Dependencies conflict with other packages
  • Dependencies add 30 seconds to every pip install
  • Dependencies don't work in air-gapped environments

So I made a rule: stdlib only. Every tool I build must work with nothing more than what ships with Python 3.8+.

This constraint became a superpower. Pure stdlib means:

  • Instant install — pip install evolver-tools completes in under 2 seconds
  • Works everywhere — Docker, CI, offline, corporate proxies
  • Never breaks — no version conflicts, ever
