1Installation

evolver-tools is a single Python package on PyPI. Install it globally or in a virtual environment — your choice. Zero dependencies means it installs in seconds, not minutes.

# Install globally (recommended — tools are meant for terminal use) $ pip install evolver-tools

That's it. Once installed, you have immediate access to all 260 tools via the evtool command.

No-pip option: try.sh

Don't want to install anything? The try.sh script downloads and runs tools on the fly — no changes to your system, no Python packages added.

# Try evolver-tools without installing $ curl -sL https://evolver-dev.github.io/evolver-tools/try.sh | bash

Docker option

Prefer containers? Run evolver-tools in a Docker container with all tools pre-installed:

$ docker run --rm -it ghcr.io/evolver-dev/evolver-tools
💡 Tip: virtual environment or global?

If you use CLI tools frequently, install globally (pip install evolver-tools). If you prefer isolation, use pipx install evolver-tools for a clean, isolated install that's still available everywhere in your terminal.

3Shell Setup (Optional but Recommended)

These small tweaks make evolver-tools feel native to your terminal. Add them to your ~/.bashrc (bash), ~/.zshrc (zsh), or ~/.config/fish/config.fish (fish).

Shorthand alias

Type et instead of evtool:

# Add to ~/.bashrc or ~/.zshrc alias et='evtool'

Shell completion

Enable tab completion for tool names:

# Bash eval "$(evtool completion bash)" # Zsh eval "$(evtool completion zsh)" # Fish eval "$(evtool completion fish)"

Use evtool as a fallback

Pipe output from other commands into evolver-tools for instant formatting:

# Pretty-print any JSON $ curl -s https://api.github.com/repos/evolver-dev/evolver-tools | evtool json-pretty # Generate a QR code from clipboard content $ cat data.txt | evtool qrcode
💡 Tip: first run message

The first time you run evtool, you'll see a welcome message explaining the project and how to support it. This only shows once.

46 Real-World Workflows

Here are practical workflows that show what evolver-tools can do. Copy and paste any of these into your terminal.

📊

1. Analyze a CSV file in one command

Get column statistics, detect data types, and generate a terminal chart — no pandas, no Jupyter, no spreadsheet app.

$ evtool csv-stats sales.csv | head -20 Column: revenue Type: float Count: 1,234 Unique: 890 Mean: $342.15 Median: $127.50 Min: $0.99 Max: $12,450.00 Top values: $9.99, $19.99, $49.99
🖥️

2. Real-time system monitor

Full TUI dashboard showing CPU, memory, disk I/O, network, and processes — no htop, btm, or glances needed.

$ evtool sysmon # Opens an interactive dashboard. Press q to quit.
🌐

3. Network diagnostics in one line

Check DNS resolution, SSL certificate expiry, and port availability — no dig, curl, or openssl needed.

$ evtool dns-lookup example.com && evtool ssl-check example.com DNS: 93.184.216.34 (A record, TTL: 86400) SSL: Valid until 2026-08-01 (60 days remaining)
🔐

4. Generate and share a secure password

Create a cryptographically secure password and output it as a QR code you can scan with your phone.

$ evtool gen-password 24 | evtool qrcode > password.png # Opens password.png with a scannable QR code in your terminal
📋

5. JSON API debugging

Fetch JSON from an API, pretty-print and validate it, then extract specific fields — all in one pipeline.

$ curl -s https://api.github.com/repos/evolver-dev/evolver-tools | evtool json-pretty | head -30 { "name": "evolver-tools", "description": "260 CLI tools...", "stars": 0, "language": "Python" }
🧹

6. Project health check

Scan a codebase for missing metadata, file size issues, broken symlinks, and license problems.

$ cd /path/to/your/project && evtool project-doctor ✓ README.md exists ✓ LICENSE file present ⚠ No .gitignore found ✓ 4 Python files — all importable

5Tips & Tricks

⚡ Pipe everything

Most tools work with stdin/stdout piping. cat data.csv | evtool csv-stats works the same as evtool csv-stats data.csv. Chain tools together for powerful one-liners.

🔍 Search before you ask

Not sure which tool does what? evtool search <keyword> fuzzy-searches all tool names and descriptions. evtool help <name> shows full usage.

📦 No version conflicts

Since all tools use only Python standard library, evolver-tools never conflicts with your other Python packages. Safe to install globally, safe for CI/CD pipelines.

🔄 Use instead of common tools

evtool replaces: htop → evtool sysmon, jq → evtool json-pretty, figlet → evtool figlet, qr → evtool qrcode, pwgen → evtool gen-password. One install, zero conflicts.
