# Launch Drafts

These are copy-paste ready drafts for distribution channels. Each is tailored to its platform.

## Available Drafts

| Platform | File | Type |
|----------|------|------|
| Hacker News (Show HN) | [show-hn-draft.md](show-hn-draft.md) | Product showcase |
| Reddit (r/commandline) | [reddit-draft.md](reddit-draft.md) | Community post |
| Dev.to | [devto-draft.md](devto-draft.md) | Technical article |

## How to Launch

1. **Post to Show HN** — https://news.ycombinator.com/submit
   - Copy `show-hn-draft.md` → paste title + body
   - Best time: weekday morning (US Eastern)

2. **Post to Reddit** — https://www.reddit.com/r/commandline/submit
   - Copy `reddit-draft.md` → paste as Text post
   - Also cross-post to: r/python, r/devops, r/selfhosted

3. **Post to Dev.to** — https://dev.to/new
   - Copy `devto-draft.md` → paste (use Markdown mode)
   - Tags: cli, python, opensource, devtools

## Tips

- **Space the posts out** — don't post all three in one day. HN first, then Reddit, then Dev.to.
- **Engage with comments** — replies from the creator (Owner) matter more than the post itself.
- **Post the story separately** — the "AI built this" angle works better as a follow-up than in the main post.
