How to run this yourself

1 Install: pip install evolver-tools
2 Fetch HN data: curl -s https://api.hnpwa.com/v0/news/1.json > hn.json
3 Convert to CSV: cat hn.json | python3 -c "import json,sys,csv,io;d=json.load(sys.stdin);buf=io.StringIO();w=csv.writer(buf);w.writerow(['rank','points','comments','title']);[w.writerow([i,item.get('points',0),item.get('comments_count',0),item.get('title','')[:50]]) for i,item in enumerate(d[:30],1)];open('hn.csv','w').write(buf.getvalue())"
4 Analyze: evtool csv-stats hn.csv — or evtool chart-cli bar hn.csv:points
