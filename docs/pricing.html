Frequently Asked Questions

Is evolver-tools really free?

Yes! All 260+ CLI tools are MIT-licensed open source. You can use them personally or commercially without any payment. Sponsorship and enterprise plans are for those who want to support development or need custom features.

How is payment handled?

Sponsorship via GitHub Sponsors — one-time or monthly. Enterprise plans are available on request via GitHub Issues.

What do enterprise plans include?

Enterprise plans cover custom tool development tailored to your workflow, priority bug fixes, dedicated support, and optional white-label deployment. Contact us via GitHub issues to discuss your needs.

Can I contribute instead?

Absolutely! The project is open source on GitHub. Star the repo, submit issues, contribute code, or help with documentation. Every contribution helps the project grow.
