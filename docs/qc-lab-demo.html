场景：黑粉浸出液钴含量测定

电池回收产线（预处理→浸出→萃取→结晶）中，浸出液是关键的中间产物。QC 化验员每天需要测定浸出液中 Co、Ni、Cu、Mn、Fe 等元素的含量，判断是否达到萃取工序的进料要求。

EDTA 滴定计算 qc-calc 预设数据 — 可调
85.25 g/L
Co(钴) 浓度计算过程：
C = 18.50 mL × 0.0500 mol/L × 58.933 g/mol / 2.00 mL
= 85.25 g/L (未稀释)
