Why evolver-tools?

260 tools isn't just a number. It's coverage. Every category covers real terminal needs.

📦Zero dependencies

Pure Python stdlib. No npm, no cargo, no apt-get. One pip install and you're done. Works in a bare Docker container.

⚡Blazing fast

No framework overhead. Each tool is a single Python file that starts and runs in milliseconds. Piped together effortlessly.

🔍Discoverable

Built-in evtool list, evtool search, evtool categories. Never forget what you have installed.

🧩Pipe-friendly

Every tool reads stdin and writes stdout. They chain together the Unix way — no learning curve.

🛠️18 categories

CSV, JSON, network, devops, security, encoding, ASCII art, generators, monitoring — your entire terminal toolkit in one package.

🆓Open source

MIT licensed. Contribute, fork, customize. The tools are yours.
