Utilities

search

Fuzzy search across all 254+ evolver-tools by name and description. Quickly find the right CLI tool for your task.

Usage

$ evtool search <query>

Flags

FlagDescription
querySearch term to find matching tools by name or description

Examples

$ evtool search csv
# Results: csv-stats, csv-filter, csv-sort, csv-pretty, csv-validate, ...
$ evtool search json
# Results: json-pretty, json-diff, json-sort, json2csv, jsonql, ...
$ evtool search network
# Results: net-analyzer, net-speed, dns-lookup, portcheck, ssl-check, ...

Details

The search tool provides fuzzy matching across all tool names and their descriptions in evolver-tools. It's the fastest way to discover what tools are available and find the right one for your task without browsing the full list.
