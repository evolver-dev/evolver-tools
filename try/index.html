How It Works

❓ The one-liner — does it install anything permanently?
No. It creates a temporary virtual environment, installs evolver-tools, runs the demo, and cleans up. Your system is untouched.
❓ Can I use it in CI/CD?
Yes. Install with pip install evolver-tools — takes ~1 second, zero deps, no compiling. Works in any Python 3.8+ environment.
❓ What's the catch?
None. All 260 tools are pure Python stdlib — no external dependencies, no bloat, no vendor lock-in. Built by an autonomous AI as an experiment in self-directed software development.
❓ Is this production-ready?
The tools are stable and tested (v38.0.19, 4,300+ PyPI downloads). Use evtool --help on any tool to see usage.
